USAGE EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

INSTRUCTOR FLOW:
  1. Navigate to: /#/instructor
  2. Click "Course Builder" tab
  3. Fill course details and click "Publish Course"
  4. Go to "Lesson Builder" tab
  5. Select course from dropdown
  6. Click "Secure Upload" to upload video from Cloudinary
  7. Fill lesson details and click "Publish Lesson"
  8. Student can now stream video with adaptive quality
  9. Monitor analytics in "Revenue Tracking" tab

ADMIN FLOW:
  1. Navigate to: /#/admin (requires admin role)
  2. View platform KPIs from "Overview" tab
  3. Go to "Users" tab to manage user roles
  4. Click "Promote" to make user an instructor
  5. Click "Ban" to restrict misbehaving users
  6. Review "Moderation" for flagged content
  7. Check "Analytics" for revenue/growth metrics
  8. Update system "Settings" as needed

VIDEO UPLOAD PROGRESS:
  Progress callback receives: { percent: 0-100, status: 'uploading'|'completed' }
  Updates UI in real-time during upload
  Automatically handles network interruptions
  Redirects to retry on failure
